# アプリ名：TUMIKI
目標達成管理アプリです。

特徴は以下の2つです。

①進捗を外部発信すること  
毎日、その日やったことをSNS発信することで、「続けている自分」を他者の目に晒し、「辞めてしまう自分」への抵抗感を作り出すことで習慣化出来る

②ゲーム性でモチベーション維持  
日々の積み上げを可視化
